# 🚀 API Automation Framework

## 📖 Project Overview

This project is an API Automation Framework developed using Java and Rest Assured.
It is designed to automate REST API testing with a structured and reusable approach.

## 🛠️ Tech Stack

* Java
* Rest Assured
* TestNG
* Maven

## ✨ Key Features

* Automated testing for REST APIs (GET, POST, PUT, DELETE)
* Reusable framework design using BaseTest class
* Configuration management using ConfigReader
* JSON response validation
* Clean and modular project structure

## 📂 Project Structure
src/
 ├── main/java/utils/ConfigReader.java
 ├── test/java/base/BaseTest.java
 ├── test/java/api/UserAPI.java
 ├── test/java/tests/UserTest.java
 ├── test/resources/config.properties


## ▶️ How to Run the Project

1. Clone the repository
2. Open in IntelliJ IDEA
3. Run `UserTest.java` or `testng.xml`
